4.1 学生端​

#4.1.1 登录​

  • 接口地址：/api/user/login
  • 请求参数：
{
    "userName": "student",  //用户名
    "password": "",  //密码
    "remember": false  //下次自动登录
}
  • 返回参数：
{
        "userName": "student",  //用户名
        "imagePath": "",  //头像
}

#4.1.2 注册​

  • 接口地址：/api/student/user/register
  • 请求参数：
{
    "userName": "student5", //用户名
    "password": "123456",  //密码
    "userLevel": 1  //年级
}
  • 返回参数：
{
    "code": 1,
    "message": "成功",
    "response": null
}
