### [#](https://www.mindskip.net:999/guide/student.html#_4-1-1-%E7%99%BB%E5%BD%95)4.1.1 登录

-   接口地址：/api/user/login
-   请求参数：

```
{
    "userName": "student",  //用户名
    "password": "",  //密码
    "remember": false  //下次自动登录
}
```

-   返回参数：

```
{
        "userName": "student",  //用户名
        "imagePath": "",  //头像
}
```
